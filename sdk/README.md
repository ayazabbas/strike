# Strike SDK

SDKs for building trading bots and integrations on [Strike](https://github.com/ayazabbas/strike) prediction markets.

Strike is a fully on-chain prediction market protocol on BNB Chain. Traders buy and sell binary outcome tokens (YES/NO) on whether an asset's price will be above or below a strike price at expiry. Uses Frequent Batch Auctions (FBA) for fair price discovery.
